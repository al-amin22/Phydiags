# User Guide

## Untuk Admin

### Masuk ke Sistem

1. Buka halaman login.
2. Masukkan kredensial admin.
3. Login ke dashboard admin.

### Mengelola Guru dan Siswa

Admin dapat:

- menambahkan akun guru
- menambahkan akun siswa
- melihat data guru
- melihat data siswa
- membuka panduan guru dan siswa

## Untuk Guru

### Melengkapi Profil

1. Masuk ke area guru.
2. Buka menu profil.
3. Isi data pribadi dan unggah foto.
4. Simpan profil.

### Membuat Kelas

1. Buka menu kelas.
2. Klik tambah kelas.
3. Isi nama kelas dan deskripsi.
4. Simpan data.

### Membuat Paket Soal

1. Buka menu paket soal.
2. Tambahkan paket soal baru.
3. Masuk ke paket soal untuk menambahkan soal satuan dan soal bertingkat.

### Membuat Ujian

1. Buka menu ujian.
2. Pilih kelas dan paket soal.
3. Isi nama ujian dan deskripsi.
4. Simpan ujian.
5. Peserta ujian dibuat otomatis dari anggota kelas.

### Melihat Hasil

Guru dapat melihat:

- hasil ujian per kelas
- hasil ujian per siswa
- hasil ujian per soal
- grafik miskonsepsi
- detail kategori miskonsepsi
- ekspor ke Excel

## Untuk Siswa

### Melengkapi Profil

1. Masuk ke area siswa.
2. Buka menu profil.
3. Isi data diri dan unggah foto.
4. Simpan profil.

### Bergabung ke Kelas

1. Buka menu kelas.
2. Masukkan kode kelas yang diberikan guru.
3. Klik gabung.

### Mengikuti Ujian

1. Buka menu ujian.
2. Pilih ujian yang tersedia.
3. Jawab setiap soal secara bertahap.
4. Simpan jawaban.
5. Selesaikan ujian setelah semua soal dijawab.

### Membuka Remediasi

Setelah ujian, siswa dapat membuka materi remediasi yang sesuai dengan hasil kesalahan, seperti:

- false positive
- false negative
- lack of knowledge
- miskonsepsi tertentu

## Panduan Materi Informasi

- gunakan halaman fungsi Phydiags untuk memahami tujuan sistem
- gunakan halaman tentang Phydiags untuk melihat gambaran umum
- baca panduan Phydiags sebelum menggunakan fitur utama
- lihat halaman perancang untuk informasi tambahan tentang sistem

## Masalah Umum

### Tidak bisa login

- periksa email dan password
- pastikan akun sesuai role

### Kelas tidak muncul

- pastikan guru sudah membuat kelas
- pastikan siswa sudah bergabung ke kelas

### Ujian tidak tersedia

- pastikan guru sudah membuat ujian
- pastikan profil pengguna sudah lengkap

### Hasil tidak tampil

- pastikan jawaban sudah tersimpan
- periksa apakah status ujian sudah selesai
